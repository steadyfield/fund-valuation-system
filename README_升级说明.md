## 🔧 故障排查

### 问题1：网站显示"暂无数据"
**解决方案：**
1. 检查GitHub Actions是否运行成功
2. 查看仓库中是否有 `data/funds.json` 文件
3. 手动触发Actions运行

### 问题2：数据不更新
**解决方案：**
1. 检查GitHub Actions运行日志
2. 确认是否在交易时间段
3. 检查requirements.txt是否包含所有依赖

### 问题3：部分基金无估值
**原因：**
- QDII基金有时差
- 债券基金波动小
- 新发基金数据源未收录

**解决方案：** 系统会自动使用备用方法

## 📞 常见问题

**Q: 为什么我的数据和天天基金网有几分钟延迟？**  
A: GitHub Actions每30分钟运行一次，所以最多有30分钟延迟。

**Q: 能否获取分钟级实时数据？**  
A: GitHub Actions有运行次数限制，建议保持30分钟间隔。

**Q: 持仓数据多久更新一次？**  
A: 持仓数据来自基金季报，每季度更新一次（1月、4月、7月、10月）。
