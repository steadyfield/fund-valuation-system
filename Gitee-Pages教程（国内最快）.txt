【部署步骤】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 1：注册 Gitee 账号
  1. 访问：https://gitee.com
  2. 点击右上角"注册"
  3. 填写信息完成注册
  4. 验证邮箱

步骤 2：导入 GitHub 仓库
  1. 登录 Gitee
  2. 点击右上角 "+" → "从 GitHub/GitLab 导入仓库"
  3. 授权 GitHub 账号
  4. 选择 "fund-valuation-system" 仓库
  5. 点击"导入"
  6. 等待导入完成

步骤 3：开启 Gitee Pages
  1. 进入导入的仓库
  2. 点击顶部的"服务" → "Gitee Pages"
  3. 选择部署分支："main"
  4. 选择部署目录："/"
  5. 点击"启动"按钮
  6. 等待部署完成（约 1-2 分钟）

步骤 4：访问网站
  - 部署成功后会显示访问地址：
    https://你的用户名.gitee.io/fund-valuation-system
  - 点击链接访问
  - 应该能看到完整的基金估值界面


【自动同步 GitHub】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法 1：手动同步（免费）
  1. 进入 Gitee 仓库
  2. 点击"同步"按钮
  3. 选择"强制同步"
  4. 等待同步完成
  5. 重新部署 Gitee Pages

方法 2：自动同步（需要 Gitee Pages Pro）
  - 升级到 Gitee Pages Pro
  - 开启自动部署
  - GitHub 更新后自动同步

方法 3：使用 GitHub Actions 自动推送
  - 配置 GitHub Actions
  - 自动推送到 Gitee
  - 触发自动部署


【配置自动推送（推荐）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在 GitHub Actions 中添加自动推送到 Gitee：

1. 在 Gitee 生成私人令牌：
   - 设置 → 私人令牌
   - 生成新令牌
   - 勾选 "projects" 权限
   - 复制令牌

2. 在 GitHub 添加 Secret：
   - Settings → Secrets and variables → Actions
   - 点击 "New repository secret"
   - Name: GITEE_TOKEN
   - Value: 粘贴 Gitee 令牌
   - 点击 "Add secret"

3. 修改 GitHub Actions 配置
   （我会帮你创建配置文件）
