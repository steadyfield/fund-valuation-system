【项目简介】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
这是一个完全免费的基金实时估值系统，利用 GitHub 的免费服务实现：
✓ 自动数据抓取（GitHub Actions）
✓ 网站托管（GitHub Pages）
✓ 零服务器费用
✓ 自动化运维


【核心功能】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 实时估值显示
   - 显示基金当前估值涨跌幅
   - 自动标注涨跌颜色（红涨绿跌）
   - 显示最后更新时间

2. 重仓股穿透分析
   - 展示前五大重仓股
   - 显示持仓比例
   - 帮助了解基金投资方向

3. 自动化更新
   - 工作日每小时自动抓取（9:00-15:00）
   - 无需手动操作
   - 数据自动同步到网站

4. 响应式设计
   - 完美支持电脑、平板、手机
   - 深色主题，护眼舒适
   - 流畅的交互体验

【部署流程】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 1 步：配置
  运行 setup.bat 或手动修改 package.json 中的用户名

第 2 步：上传
  使用 Git 将代码上传到 GitHub

第 3 步：部署
  在 GitHub 开启 Pages 并触发 Actions

第 4 步：访问
  通过 https://你的用户名.github.io/fund-valuation-system 访问


【自定义指南】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 修改监控基金列表
   编辑：scripts/fetch_data.py
   找到：WATCH_LIST = {...}
   添加：你的基金代码和名称

2. 修改网站样式
   编辑：src/index.css
   修改：颜色、字体、布局等

3. 修改更新频率
   编辑：.github/workflows/update_data.yml
   修改：cron 表达式

4. 添加新功能
   编辑：src/App.js
   添加：你想要的功能组件


【数据说明】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

更新频率：工作日每小时（9:00-15:00）
数据内容：
  - 基金实时估值
  - 估值涨跌幅
  - 前五大重仓股
  - 持仓比例

注意事项：
  ⚠️ 数据仅供参考，不构成投资建议
  ⚠️ QDII 基金可能没有实时估值
  ⚠️ 重仓股数据来自最近一期报告
