只需要做一次：

1. 提交代码到 GitHub
   双击运行：部署全网最丰富版本.bat

2. 开启 GitHub Actions
   - 进入 GitHub 仓库
   - 点击 Actions 标签
   - 点击 "I understand my workflows, go ahead and enable them"

3. 手动触发首次运行
   - 点击 "Auto Fund Valuation & Deploy"
   - 点击 "Run workflow"
   - 等待运行完成

4. 完成！
   - 之后完全自动运行
   - 无需任何手动操作
