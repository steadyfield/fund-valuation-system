  General  ← 你现在在这里

  ├─ Code and automation

步骤 2：配置设置
  进入 Pages 页面后，你会看到：
  
  ┌─────────────────────────────────────┐
  │ Build and deployment                │
  │                                     │
  │ Source:                             │
  │ [Deploy from a branch ▼]            │
  │                                     │
  │ Branch:                             │
  │ [main ▼] [/ (root) ▼] [Save]       │
  └─────────────────────────────────────┘
  
  选择：
  - Source: Deploy from a branch
  - Branch: main
  - Folder: / (root)
  - 点击 Save

步骤 3：查看结果
  保存后，页面顶部会显示：
  
  ┌─────────────────────────────────────┐
  │ ✓ Your site is live at              │
  │ https://你的用户名.github.io/...     │
  └─────────────────────────────────────┘


【如果看不到 Pages】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

原因 1：仓库是私有的
  解决：
  1. 在 General 页面向下滚动
  2. 找到 "Danger Zone"
  3. 点击 "Change visibility"
  4. 选择 "Make public"
  5. 确认
  6. 刷新页面，Pages 就会出现

原因 2：浏览器问题
  解决：
  1. 刷新页面（F5）
  2. 或清除缓存后刷新（Ctrl + F5）
